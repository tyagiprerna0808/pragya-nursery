# Pragya Nursery

Online plant store for Greater Noida. Curated catalogue, same-day delivery, paid checkout.

[pragyanursery.co.in](https://www.pragyanursery.co.in)

## About

Pragya Nursery sells live plants across Greater Noida. The catalogue spans
indoor, outdoor, flowering, fruit, and exotic varieties — each one nursery-grown,
inspected before dispatch, and delivered within a 15 km radius the same day
it is ordered. Online checkout end to end; in-store pickup also available.
